# Módulo 2: Comandos Essenciais

Este diretório contém os scripts de exemplo e exercícios práticos do Módulo 2 do curso.

## Estrutura

```
module2/
├── navigation/
│   └── nav_demo.sh        # Demonstração de comandos de navegação
├── file-ops/
│   └── file_ops.sh        # Operações com arquivos
├── text-proc/
│   └── text_tools.sh      # Processamento de texto
└── system/
    └── sys_info.sh        # Informações do sistema
```

## Como usar

1. Clone este repositório
2. Dê permissão de execução aos scripts:
   ```bash
   chmod +x */*.sh
   ```
3. Execute os scripts individualmente:
   ```bash
   ./navigation/nav_demo.sh
   ```

## Exercícios

Cada diretório contém scripts que demonstram os conceitos abordados em cada tópico do módulo.
Experimente modificar os scripts e criar suas próprias versões para praticar.
